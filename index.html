This repo does not have any HTML content and is intended to be used to add the Session repository inside the F-Droid app.
Please add this repository by going to Settings > Repositories > + Button and entering this URL (https://fdroid.getsession.org) inside the F-Droid app.
Or by clicking this deep link if you have F-Droid installed Once added you can search for the Session app in the F-Droid store
